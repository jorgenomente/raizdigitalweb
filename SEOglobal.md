SISTEMA SEO GLOBAL — Cosmic Studio (versión unificada y optimizada)

Aplicar estas reglas a TODAS las páginas, servicios, landings y futuros contenidos.
Garantiza coherencia, posicionamiento estable y máxima compatibilidad con Google.

1. URL perfectas (claras, cortas y con keyword)

Formato:
/{categoria}/{keyword-principal}

Ejemplos:

/servicios/branding-profesional

/servicios/diseño-web

/servicios/marketing-digital

❌ Evitar: /servicio1, /abc123, /varios, /misc.

Regla Global: cada URL debe decir exactamente de qué trata la página.

2. Meta Title (50–60 caracteres)

Incluye keyword principal + marca.

Plantilla:
{KEYWORD_PRINCIPAL Capitalizada} | Cosmic Studio

Ejemplos:

Branding Profesional | Cosmic Studio

Diseño Web Profesional | Cosmic Studio

Marketing Digital Estratégico | Cosmic Studio

Regla Global: cada página tiene un Meta Title único y optimizado.

3. Meta Description (150–160 caracteres)

Debe tener keyword + beneficio claro.

Plantilla:
{KEYWORD_PRINCIPAL Capitalizada} para emprendedores y negocios. Soluciones profesionales que elevan tu presencia, comunican mejor y generan confianza.

Regla Global: siempre incluir beneficios, nunca solo texto descriptivo.

4. H1 único obligatorio

Debe repetir la keyword principal una sola vez.

Plantilla:
{KEYWORD_PRINCIPAL Capitalizada} Profesional

Ejemplos:

Branding Profesional

Diseño Web Moderno

Marketing Digital Estratégico

Regla Global: Nunca usar más de un H1 por página. Google penaliza el multi-H1.

5. Estructura de títulos (jerarquía universal)

Aplicar SIEMPRE esta jerarquía:

H1 — Keyword principal
H2 — Beneficios
H2 — Qué incluye este servicio
H2 — Para quién es
H2 — Nuestro proceso / Cómo funciona
H2 — Puntos de dolor (opcional, recomendado)
H2 — Preguntas frecuentes (opcional)
H2 — CTA final o cierre emocional

Regla Global: Los H2 deben ordenar el contenido. Nada de H3 como títulos principales.

6. ALT text SEO para imágenes

Usar keyword + descripción funcional.

Plantillas:

Ejemplo de {KEYWORD_PRINCIPAL} diseñado por Cosmic Studio

{KEYWORD_PRINCIPAL}: diseño moderno y profesional

Regla Global: Cada página debe tener al menos 1 imagen con ALT optimizado.

7. Rich Snippet JSON-LD (obligatorio en todos los servicios)

Plantilla universal:

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Service",
  "name": "{KEYWORD_PRINCIPAL Capitalizada}",
  "provider": {
    "@type": "Organization",
    "name": "Cosmic Studio",
    "url": "https://cosmicst.dev"
  },
  "description": "Servicio de {KEYWORD_PRINCIPAL} profesional para emprendedores, negocios y marcas digitales.",
  "areaServed": "Argentina, Latinoamérica, Estados Unidos",
  "serviceType": "{KEYWORD_PRINCIPAL Capitalizada}"
}
</script>


Regla Global: Todas las páginas de servicio deben incluir su JSON-LD correspondiente.

8. Links internos obligatorios en cada página

Siempre incluir enlaces hacia:

Portfolio

Servicio Web

Servicio Branding

Servicio Marketing

Regla Global: El interlinking aumenta el posicionamiento general del sitio.

9. Core Web Vitals (para TODA la web)

✔ Imágenes en .webp o .avif
✔ Lazy loading obligatorio
✔ Usar <Image /> de Next.js
✔ Fuentes con font-display: swap
✔ Evitar videos pesados sin optimizar

Regla Global: Nada que afecte velocidad puede publicarse sin optimizar.

10. Contenido fresco (mantener + actualizar)

Actualizar cada 6–12 meses:

Ejemplos

Fechas

Casos de estudio

Procesos internos

Capturas de pantalla

Resultados

Portfolio

Regla Global: Google penaliza páginas estáticas. Refrescar aumenta ranking.

11. Contenido 100% único

No duplicar:

Meta Title

Meta Description

H1

Párrafos clave

Estructura de texto

Regla Global: Cada servicio debe sentirse como una pieza única.

12. Densidad de la keyword bien distribuida

La keyword principal debe aparecer:

Meta Title

Meta Description

H1

Primer párrafo

1–2 H2

Un ALT

❌ No repetir más de eso.

Regla Global: Evitar sobreoptimizar y generar penalización
